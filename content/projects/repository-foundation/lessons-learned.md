# Durable lessons

## The archive must stand on its own

If a file stops making sense outside the site, the content model has drifted in the wrong direction.

## Derived content is disposable

Social posts can be regenerated. Decision records, logs, failures, and observations cannot.

## Predictable templates beat clever flexibility

Regular sections make authoring easier for humans and more useful for AI transformation.

## Migration safety should be visible in the tree

The folder structure should communicate which parts are canonical and which are implementation detail.
