# External family reference

An official Comau compact robot catalogue from 2023 lists the model `NS-16-1.65` as a 6-axis robot with:

- 16 kg wrist payload
- 1650 mm reach
- 333 kg robot weight
- floor, ceiling, and sloped mounting options
- IP65 and IP67 foundry-protected variants

Source:

- Comau Robotics Compact Catalogue 2023: https://www.comau.com/wp-content/uploads/2023/06/Comau_Robotics_Compact-Catalogue_EN.pdf

# Important caution

This is family-reference material only. It is useful for orientation, but it must not be treated as a confirmed specification sheet for this exact project machine until the robot nameplate, manual set, or serial-specific documentation has been checked.
