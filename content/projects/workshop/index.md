# Workshop

The workshop is not just background context for other projects. It is an operating environment that shapes what can be built, repaired, tested, and stored.

## Focus

- tool organisation that reduces friction for repeat work
- workbench and machine layout that supports fabrication and repair
- electrical, storage, and safety improvements that increase usable capacity
- documentation of fixtures, jigs, and repeatable setup patterns

## Why archive it

Physical working environments tend to evolve through small changes that are later forgotten. Treating the workshop as a project preserves decisions that affect every build after it.
